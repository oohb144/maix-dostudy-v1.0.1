# 语音控制关键词文档

本文档汇总本系统支持的全部语音关键词，**以 `config.py` 中的 `VOICE_KEYWORDS`（离线）与 `ONLINE_VOICE_KEYWORDS`（在线）为准**。修改关键词请同步更新本文件。

> 说明：
> - **离线识别**使用拼音关键词（声调数字紧跟音节，如 `shi2 bie2` = 识别），由本机语音模型处理。
> - **在线识别**（DashScope）使用中文文本子串匹配，可一条命令对应多个说法。
> - 所有命令统一由 `main.py::_on_voice_command` 派发，命令名（value）在两套表里保持一致。

---

## 一、离线关键词（`VOICE_KEYWORDS`）

| 拼音 | 中文含义 | 命令名 | 功能 |
|------|---------|--------|------|
| `zhu3 jie4 mian4` | 主界面 | `home` | 返回主页 |
| `she4 zhi4` | 设置 | `settings` | 打开设置页 |
| `lu4 ru4` | 录入 | `enroll` | 进入人脸录入 |
| `lu4 xiang4` | 录像 | `recordings` | 打开录像列表 |
| `shi2 bie2` | 识别 | `recognize` | 开始识别 |
| `ting2 zhi3` | 停止 | `stop` | 停止当前操作（识别/录制/录入） |
| `kai1 shi3` | 开始 | `start` | 开始（等同开始识别） |
| `da4 kai1 lu4 zhi4` | 打开录制 | `auto_record_on` | 开启自动录制 |
| `guan1 bi4 lu4 zhi4` | 关闭录制 | `auto_record_off` | 关闭自动录制 |
| `du2 qu3 xin4 xi1` | 读取信息 | `read_info` | 语音播报系统信息 |
| `xian4 shi4 xin4 xi1` | 显示信息 | `show_info` | 显示系统信息 |
| `bo1 fang4 yin1 pin3` | 播放音频 | `play_audio` | 播放音频文件 |
| `ting2 zhi3 bo1 fang4` | 停止播放 | `stop_audio` | 停止音频播放 |

## 二、在线关键词（`ONLINE_VOICE_KEYWORDS`）

在线识别采用“中文文本子串 → 命令名”匹配，同一命令可配置多个说法。

### 1. 界面切换
| 中文说法 | 命令名 | 功能 |
|---------|--------|------|
| 主界面 / 回到主页 / 主页 | `home` | 返回主页 |
| 设置 | `settings` | 打开设置页 |
| 录入 | `enroll` | 进入人脸录入 |
| 录像 | `recordings` | 打开录像列表 |

### 2. 播放 / 融合界面
> 具体词在前，避免“播放”被通用匹配。

| 中文说法 | 命令名 | 功能 |
|---------|--------|------|
| 进入播放界面 / 播放界面 / 进入播放 | `fusion_page` | 进入融合播放界面 |
| 开始融合 / 融合视频 / 融合 | `fusion_mux` | 开始融合视频 |
| 播放录像 / 播放视频 / 开始播放 | `fusion_play` | 播放选中的融合视频 |

### 3. 音频播放
| 中文说法 | 命令名 | 功能 |
|---------|--------|------|
| 播放音频 | `play_audio` | 播放音频文件 |
| 停止播放 | `stop_audio` | 停止音频播放 |

### 4. 功能控制
| 中文说法 | 命令名 | 功能 |
|---------|--------|------|
| 开始识别 | `recognize` | 开始识别 |
| 打开自动录制 / 开启自动录制 / 打开录制 | `auto_record_on` | 开启自动录制 |
| 关闭自动录制 / 关闭录制 | `auto_record_off` | 关闭自动录制 |
| 停止 | `stop` | 停止当前操作 |
| 开始 | `start` | 开始（等同开始识别） |

### 5. 信息查询
| 中文说法 | 命令名 | 功能 |
|---------|--------|------|
| 读取信息 | `read_info` | 语音提示"请查看屏幕"，同时在屏幕显示系统信息浮层 3 秒 |
| 显示信息 | `show_info` | 在屏幕绘制系统信息浮层 3 秒（当前状态 + 已录入人数，无语音） |

## 三、命令功能说明

| 命令名 | 触发动作 |
|--------|---------|
| `home` | 返回主页（清空页面栈到主页） |
| `settings` | 跳转到设置页 |
| `enroll` | 跳转到录入页并进入录入状态 |
| `recordings` | 跳转到录像列表页 |
| `fusion_page` / `fusion_mux` / `fusion_play` | 融合播放相关，仅在播放页生效 |
| `recognize` / `start` | 空闲态进入识别态 |
| `stop` | 识别/录制/录入态退回空闲态 |
| `auto_record_on` | 打开自动录制；识别态下重置人脸类型，下次检测到人脸即转录制 |
| `auto_record_off` | 关闭自动录制；录制中会退回识别态，开始按熟人/陌生播音效 |
| `read_info` | 语音提示看屏幕 + 屏幕显示信息浮层 3 秒（当前状态与已录入人数） |
| `show_info` | 屏幕绘制信息浮层 3 秒（当前状态与已录入人数，无语音） |
| `play_audio` | 播放 `AUDIO_DIR` 下首个音频文件 |
| `stop_audio` | 停止音频播放 |
